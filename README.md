# jokenpython
Clássico jogo pedra, papel e tesoura (jokenpo) com frases de efeito feito em python.

O arquivo "joke_colors.py" é o game feito para ser usado numa IDE (De preferência Visual Studio Code). 

Já o arquivo "joke_nocolors.py" pode ser executado tanto numa ide quanto no terminal. 

A única diferença é que no primeiro algumas palavras ou fundo dos caracteres apresentam algumas cores para dar um pouco mais de graça ao jogo.
